How It Works

This example demonstrates Bootstrap modal dialogs with HTMX integration:

  • HTMX Integration: Uses hx-get="/modal" to load modal content dynamically
  • Bootstrap Modal: Uses data-bs-toggle="modal" to show the modal
  • Dynamic Content: Modal content is loaded from the server via HTMX
  • Targeted Loading: Uses hx-target="#modals-here .modal-content" to load content into the modal-content div
HTMX Patterns Used:
  • hx-get - Load modal content from server
  • hx-target - Target the modal container
  • hx-trigger="click" - Trigger on button click
Bootstrap Features:
  • Native modal trigger with data-bs-toggle="modal"
  • Responsive modal dialog with modal-lg modal-dialog-centered
  • Backdrop blur effect with modal-blur class
