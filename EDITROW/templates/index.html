Edit Row Example

Click the edit button on any row to make it editable. Only one row can be edited at a time.

How it works:

  • hx-target="closest tr": Targets the closest table row to the clicked button
  • hx-swap="outerHTML": Replaces the entire row with the server response
  • hx-trigger="edit": Uses custom event to trigger edit mode
  • hx-include="closest tr": Includes all form inputs from the row when saving
  • Single-instance editing: Prevents multiple rows from being edited simultaneously

HTMX Patterns demonstrated:

  • hx-get: Load edit forms and contact data
  • hx-put: Submit updated contact information
  • hx-target: Precise targeting of table rows
  • hx-swap: Smooth content replacement
  • hx-trigger: Custom event handling
  • hx-include: Form data inclusion
