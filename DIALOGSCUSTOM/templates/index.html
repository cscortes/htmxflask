This example demonstrates how to build custom modal dialogs from scratch using HTMX and Hyperscript, without relying on CSS frameworks.

How It Works

This example shows how to create custom modal dialogs with HTMX:

  • HTMX Loading: Uses hx-get="/modal" to load modal content from server
  • Append to Body: Uses hx-swap="beforeend" to append modal to end of body
  • Hyperscript Animations: Uses Hyperscript for smooth open/close animations
  • Event-Driven Close: Hyperscript triggers closeModal event to remove dialog
  • CSS Animations: Custom keyframe animations for fade and zoom effects

HTMX Patterns Used:

  • hx-get - Load modal content from server
  • hx-target="body" - Target the body element
  • hx-swap="beforeend" - Append content to end of target

Hyperscript Features:

  • _="on closeModal..." - Event-driven modal removal
  • add .closing - Add CSS class for close animation
  • wait for animationend - Wait for animation to complete
  • remove me - Remove modal from DOM

Key Features:

  • No CSS framework dependencies
  • Smooth fade-in/fade-out animations
  • Zoom animations for modal content
  • Click outside to close (underlay)
  • Close button with event trigger
  • Fully customizable styling
