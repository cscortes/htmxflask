HTMX UIKit Modal Dialogs Example

This is an example of using HTMX to remotely load modal dialogs using UIKit. The modal features enhanced CSS animations that fade in from the top with a backdrop blur effect. We use Hyperscript to demonstrate how cleanly that scripting language allows you to glue htmx and other libraries together.

How It Works

This button uses a GET request to /modal when this button is clicked. The contents of this file will be added to the DOM underneath the #modals-here DIV.

Rather than using the standard UIKit Javascript library we are using a bit of Hyperscript, which triggers UIKit's smooth animations. It is delayed by 10ms so that UIKit's animations will run correctly.

The modal features enhanced CSS animations including:

  • Fade-in from top: Modal slides down from above with opacity transition
  • Backdrop blur: Background content is blurred when modal is open
  • Smooth transitions: All animations use CSS transitions for smooth effects
  • Enhanced styling: Modern button and input styling with hover effects

Finally, the server responds with a slightly modified version of UIKit's standard modal.
