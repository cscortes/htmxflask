Delete Row Example

Click the delete button on any row to remove it with confirmation and fade-out animation.

How it works:

  • hx-confirm="Are you sure?": Shows a browser confirmation dialog before deletion
  • hx-target="closest tr": Targets the closest table row to the clicked button
  • hx-swap="outerHTML swap:1s": Replaces the entire row and waits 1 second for CSS fade-out
  • hx-delete: Sends a DELETE request to remove the contact from the server

HTMX Patterns demonstrated:

  • hx-delete: RESTful DELETE method for resource removal
  • hx-confirm: User confirmation before destructive actions
  • hx-target: Precise targeting of table rows
  • hx-swap: Controlled content replacement with timing
  • Empty responses: HTMX gracefully handles empty server responses

Animation details:

  • Fade-out effect: CSS transition reduces opacity over 1 second
  • Timing coordination: hx-swap delay matches CSS transition duration
  • Smooth removal: Row disappears after animation completes
